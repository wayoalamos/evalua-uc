### Set-Up
git clone https://github.com/wayoalamos/evalua-uc-api.git
npm install express
npm install graphql
apt-get install postgresql
npm install pg-promise

### terminal commands to remember
psql -d postgres -U me         # connect to databse
\dt                            # list all tables
\c evalua-uc-api                

sudo kill `lsof -t -i:3000`
